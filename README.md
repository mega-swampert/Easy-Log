# EasyLog

Um aplicativo que centraliza e organiza todas as informações de uma marcenaria, permitindo gestão eficiente de estoque, projetos, máquinas, funcionários e fornecedores, substituindo todos os processos em excel.

# Principais Dores:
- Atualmente, o controle da marcenaria é feito em excel e de forma descentralizada, o que gera perda de informações, dificuldades de consulta e possui diversas oportunidades para erros.  
- A falta de um sistema único dificulta a comunicação entre todas as partes envolvidas, resultando em atrasos nas tomadas de decisão.  
- Não existe rastreabilidade clara sobre os projetos em progresso, prejudicando a consulta e desenvolvimento claro para a equipe.

# Modelo de Negócios:
- O sistema será voltado para marcenarias, tomando como base características e termos utilizados nesses negócios.
- Nosso valor está em centralizar informações críticas sobre projetos, clientes e máquinas em um ambiente seguro e intuitivo.
- Expansão do sistema para mais de um negócio.

# Requisitos de Alto Nível:
- O sistema deve permitir o cadastro de novos clientes (nome, endereço, pessoa física ou jurídica);

- O sistema deve armazenar e controlar o estoque da empresa (madeira, tinta, tinner, EPIs, etc);

- O sistema deve registrar funcionários e reconhecer sua função;

- O sistema deve controlar a manutenção de máquinas e se ela está ou não em funcionamento;

- O sistema deve permitir o registro de novos fornecedores e atualização de existentes;

- O sistema deve vincular fornecedores as entregas realizadas;

- O sistema deve diferenciar os tipos de projetos.
